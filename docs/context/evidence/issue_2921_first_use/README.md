## Overview
This directory contains the first-use comparison report generated by `scripts/adversarial/run_proposal_vs_random_issue_2921.py`.
Since no real certified failure archive was provided, the comparison script executed under the
`diagnostic_only` state, using synthetic archive and search-space fixtures. The recorded reason is
`No archive path provided; using synthetic archive fixture. No search space path provided; using synthetic search-space fixture.`

The proposal ranking and the diagnostic objective both use archive-nearness in the synthetic
fixture. The reported deltas are therefore circular by construction and validate report plumbing
only. Held-out evidence remains `false` until a real certified archive run proves scenario-family,
seed, archive-ID, and evaluation-outcome disjointness and passes null tests.
