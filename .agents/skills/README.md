# Repo-Local Context Skills

These skills complement the execution-oriented workflows under `.codex/skills/`.

Use `.agents/skills/` when the task is mainly about gathering the right repository context,
reviewing benchmark-sensitive changes, or drafting paper-facing/project-facing docs without
re-deriving the same navigation notes each time.

Available skills:

- `benchmark-overview`
- `experiment-context`
- `planner-integration`
- `paper-facing-docs`
- `review-benchmark-change`
