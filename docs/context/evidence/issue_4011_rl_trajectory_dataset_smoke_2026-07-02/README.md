# Issue #4011 RL Trajectory Dataset Smoke Evidence

This bundle is a tiny `RLTrajectoryDataset.v1` preview generated by
`scripts/benchmark/record_rl_trajectory_dataset.py` from a synthetic benchmark episode JSONL record.
It proves the recorder can convert `algorithm_metadata.simulation_step_trace` frames with
`rl.reward`, `rl.terminated`, and `rl.truncated` into an episode-major offline reinforcement
learning trajectory row plus manifest.

Files:

- `issue_4011_smoke.preview.jsonl`: one two-step preview episode.
- `issue_4011_smoke.manifest.json`: manifest with checksum, split summaries, and provenance.

Durability boundary: this is a compact review fixture, not a durable training dataset. Full datasets
remain worktree-local under `output/` until promoted to an artifact store with an explicit URI and
checksum.
