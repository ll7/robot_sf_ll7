# Benchmark Artifact Captions

## fig_planner_status_summary

diagnostic-only count of campaign-table rows by planner status. Fallback, degraded, and not_available rows are caveats rather than benchmark-success evidence.

## tab_campaign_table

Compiled campaign table preserving planner execution status and limitation labels.

## tab_not_available_inputs

Optional compiler inputs that were absent from the campaign report directory.
